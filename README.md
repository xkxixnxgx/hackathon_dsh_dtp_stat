# Разработка сервиса по определению мест концентрации ДТП

Проект в рамках хакатона Digital Super Hero (Sep, 2020)

## Описание

Web-сервис для определения мест концентрации дорожно-транспортных происшествий (далее – ДТП).

1. **Определение мест концентрации ДТП на основе данных о произошедших ДТП за отчетный период (1 год).** <br>*Место концентрации ДТП* – это участок дороги, улицы, не превышающий 1000 метров вне населенного пункта или 200 метров в населенном пункте, либо пересечение дорог, улиц, где в течение отчетного года произошло три и более дорожно-транспортных происшествия одного вида или пять и более дорожно-транспортных происшествий независимо от их вида, в результате которых погибли или были ранены люди (из Федерального закона от 10.12.1995 No196-ФЗ (ред. от 30.07.2019) «О безопасности дорожного движения»).

>Для выполнения данной задачи предоставлены данные о ДТП за 2017, 2018, 2019 годы по Республике Татарстан (в виде таблиц формата *.xlsx).<br>
>**Примечание**: методика по ручному выявлению мест концентрации ДТП представлена в документе *ОДМ 218.6.015-2015 «Рекомендации по учету и анализу дорожно-транспортных происшествий на автомобильных дорогах Российской Федерации»* (приложение Е).

2. **Создание рекомендательной модели для оптимизации расстановки комплексов ФВФ.**<br>Модель должна на основе данных о существующем расположении комплексов ФВФ и данных о расположении мест концентрации ДТП, исходя из вида ДТП (столкновение, опрокидывание, наезд на пешехода), полученных из п.1, рекомендовать места для оптимального расположения комплексов ФВФ с целью уменьшения общей аварийности. Модель должна уметь прогнозировать изменение мест концентрации ДТП на предстоящий период. Необходимо также разработать методику оценки эффективности размещения комплексов ФВФ. Для выполнения данной задачи будут предоставлены данные о расположении стационарных и передвижных комплексов ФВФ (в формате *.csv).

3. **Разработка web-сервиса для удобной работы с полученными знаниями.**<br>Сервис должен обеспечивать разделение уровней доступа пользователей с предоставлением им различного функционала. Обычные пользователи должны иметь возможность использовать сервис без предварительной регистрации. Им предоставляется информация о расположении мест концентрации ДТП на карте Республики Татарстан по выбранному году. Также должна быть возможность посмотреть детализацию по каждому ДТП в выбранном месте. Продвинутые пользователи должны иметь возможность зарегистрироваться в личном кабинете. Наряду с функционалом обычного пользователя им предоставляется возможность проводить аналитику по выбранным местам концентрации ДТП, с возможностью загрузки новых данных, обработки новых данных, визуализации данных, публикации обработанных данных.

4. **Подключение внешних погодных и других сервисов для выявления влияния условий на возникновение ДТП.**

## Ожидаемый результат
Выявление наиболее опасных мест концентрации ДТП, а также прогноз миграции всех мест концентрации ДТП на будущий период с учетом оптимизации размещения комплексов.
